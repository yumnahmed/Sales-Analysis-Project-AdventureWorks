# 📊 Sales Analysis Project – AdventureWorks20200

## 📌 Overview

This project presents an interactive sales analysis dashboard built using Power BI, based on the AdventureWorks dataset. It provides a comprehensive view of sales performance across products, territories, and time.

The dashboard enables users to explore key business metrics, identify trends, and gain actionable insights to support data-driven decision-making.


## 🎯 Objectives

* Analyze overall orders performance
* Compare orders across different territories
* Identify top-performing products and categories
* Track sales trends over time (Order Date, Ship Date, Due Date)
* Provide insights to improve revenue and operations


## 🛠️ Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* SQL Server
* Data Modeling (Star Schema)


## 📂 Data Source

AdventureWorks Sample Database:https://learn.microsoft.com/en-us/sql/samples/adventureworks-install-configure?view=sql-server-ver17&tabs=ssms
Includes:

* Sales Orders
* Products
* Customers
* Territories
* Shipping Methods


## 🧱 Data Modeling

* Star Schema Design
* Fact Table: Sales Order Details
* Dimension Tables:

  * Products
  * Status
  * Territories
  * Dates
  * Ship Methods


## 📊 Key Features

* KPIs (Total Sales, Tax, Freight, Subtotal)
* Sales by Territory Analysis
* Time Intelligence Analysis (Year, Month, Day)
* Interactive Filters (Slicers)
* Drill Down & Drill Through
* ToolTip


## 📈 Key Insights

* Significant variation in sales across territories
* Southwest region shows the highest revenue
* Canada demonstrates strong growth potential
* Accessories category has the highest order volume
* Sales peaked in a specific year and declined afterward


## 💡 Recommendations

* Invest more in high-performing regions (e.g., Southwest, Canada)
* Improve performance in low-performing areas
* Use cross-selling strategies for high-demand products
* Analyze the decline in sales trends and adjust strategies
* Optimize logistics and shipping methods
